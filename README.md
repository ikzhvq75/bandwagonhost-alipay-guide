## 一、搬瓦工到底是个什么商家：先认识再下单

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 服务商，主打 KVM 虚拟化的自管型 VPS，自己拥有硬件设备和 IP 段，机房分布在全球十几个数据中心。它在国内用户圈子里火了这么多年，核心卖点就两条：一是 CN2 GIA / CN2 GIA-E 系列线路对国内三网回程做了优化，电信用户尤其受益；二是 KiwiVM 自研控制面板功能完整，重装系统、迁移机房、做快照、查流量都在一个面板里搞定，对新手友好。

需要注意的是，搬瓦工是 **self-managed（自管理）** 服务商——意思是服务器层面的运维你自己来，它只负责硬件和网络。这也是它能把价格压下来的原因。如果你需要 7×24 小时的技术支持代你装环境、排查故障，那要找的是 fully-managed 服务商，搬瓦工这条路不太适合你。

支付方式上，搬瓦工支持支付宝、银联、PayPal、信用卡。其中 **支付宝对国内用户来说最顺手**：扫码或登录账户直接付人民币，系统自动按汇率折算美元，退款也是原路返回。这里有个细节要提醒——账户余额充值目前不支持支付宝，只能用 PayPal 或信用卡充值余额；但直接下单购买套餐时，支付宝是可用的。所以别想着先充值再买，直接在订单页选支付宝付款就行。

## 五、机房怎么选：按你用的运营商对号入座

搬瓦工有十几个机房，按国内访问体验排序大致是这样：

- **电信用户**：DC9 CN2 GIA > DC6 CN2 GIA-E > 日本软银 JPOS_1 > 荷兰 EUNL_9 > DC3 CN2 > DC8 ZNET
